# Chess

Chess made with raylib library

## Build

* Requires [raylib](https://www.raylib.com/)
* Requires `pkg-config`

```sh
$ git clone https://github.com/diwasrimal/chess-c.git
$ cd chess-c
$ ./build.sh    # or `make`
$ ./chess
```

## Controls
* `Esc` to exit
* `R` to restart
